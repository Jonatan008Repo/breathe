# breathe

Página web interactiva con ejercicio de respiración en 4 tiempos para relajación y concentración.

## Tecnologías

- HTML5, CSS3, JavaScript (vanilla)
- Google Fonts (Montserrat)
- Google AdSense
- GitHub Pages

## Desarrollo

```bash
# Servir localmente
python -m http.server 8000 --directory docs
```

Abrir http://localhost:8000 en el navegador.

## Estructura

- `docs/` - Código fuente (HTML, CSS, JS, assets)
- `.github/prompts/` - Planes de trabajo y documentación

## Despliegue

El sitio se despliega automáticamente en GitHub Pages desde la rama `main`.
